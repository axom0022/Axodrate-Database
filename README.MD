## Commands

### Games
- `/tictactoe` - Play tic-tac-toe with another user
- `/guess` - Guess a number between 1-10
- `/mathgame` - Play a math game
- `/rps` - Play rock-paper-scissors
- `/coinflip` - Flip a coin and bet coins
- `/roll` - Roll a dice with custom max value
- `/hangman` - Play hangman
- `/poll` - Create a poll with up to 4 options

### Economy
- `/balance` - Check your or another user's balance
- `/talk` - Earn coins by talking (min 10 words)
- `/gamble` - Gamble coins
- `/give` - Give coins to another user
- `/shop` - View the shop
- `/buy` - Buy an item from the shop
- `/inventory` - View your or another user's inventory
- `/useitem` - Use an item from your inventory
- `/addcoins` - Add coins to a user (Admin only)
- `/removecoins` - Remove coins from a user (Admin only)
- `/setcoins` - Set a user's coins to a specific amount (Admin only)

### Info
- `/profile` - View your or another user's profile
- `/serverinfo` - View server information
- `/userinfo` - View user information
- `/avatar` - View your or another user's avatar
- `/leaderboard` - View the server leaderboard
- `/xp` - Check your or another user's XP and level

### Moderation
- `/warn` - Warn a user for breaking rules
- `/edit_warn` - Edit a specific warning by ID
- `/remove_warn` - Remove a specific warning by ID
- `/warnings` - View warnings for a user
- `/kick` - Kick a user from the server
- `/ban` - Ban a user from the server
- `/timeout` - Timeout a user for specified minutes
- `/untimeout` - Remove timeout from a user
- `/clear` - Clear a specified number of messages
- `/lock` - Lock the current channel
- `/unlock` - Unlock the current channel
- `/slowmode` - Set slowmode for the current channel
- `/nick` - Change a user's nickname
- `/purge` - Purge messages (max 100)

### Utilities
- `/remind` - Set a reminder
- `/timer` - Set a timer (max 5 minutes)
- `/ping` - Check bot latency
- `/embed_create` - Create an embed message
- `/embed_edit` - Edit an existing embed message
- `/test` - Test if the bot is working
- `/help` - View all commands

### Giveaways
- `/giveaway` - Create a giveaway
- `/giveaway_list` - List active giveaways
- `/giveaway_reroll` - Reroll a giveaway winner

### Tickets
- `/ticket_panel` - Create a ticket panel
- `/ticket` - Create a ticket

### Custom Commands
- `/addcustom` - Add a custom command
- `/removecustom` - Remove a custom command
- `/listcustom` - List all custom commands

### Settings
- `/enable_welcome` - Enable welcome messages
- `/disable_welcome` - Disable welcome messages
- `/enable_goodbye` - Enable goodbye messages
- `/disable_goodbye` - Disable goodbye messages
- `/enable_antispam` - Enable anti-spam protection
- `/disable_antispam` - Disable anti-spam protection
- `/enable_antilinks` - Enable anti-links protection
- `/disable_antilinks` - Disable anti-links protection
- `/enable_autocensor` - Enable auto-censorship
- `/disable_autocensor` - Disable auto-censorship
- `/disable_censor` - Disable all censorship and clear censor list
- `/censor_add` - Add a word to the censor list
- `/censor_remove` - Remove a word from the censor list
- `/censor_import` - Import multiple censored words from a file
- `/add_multicensor` - Add multiple censored words at once
- `/togglestatus` - View status of all feature toggles
- `/welcome_setup` - Setup welcome messages
- `/goodbye_setup` - Setup goodbye messages
- `/set_level_role` - Set a role to be given at a specific level
